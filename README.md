# MindLift

## Overview

The MindLift delivers daily inspiration through push notifications, goal tracking, and personalized content.

## Features

- Daily Motivational Notifications
- Goal Setting and Tracking
- Personalized Content

## Technology Stack

- Frontend: Flutter
- Backend: Firebase
